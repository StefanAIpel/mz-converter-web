# MZ Converter Web

Statische frontend voor de MZ Converter.

## Stack

- HTML
- CSS
- Vanilla JavaScript
- Supabase Auth
- Render backend API

## Backend

- `https://mz-converter-api.onrender.com`

## Local preview

```bash
python3 -m http.server 8080
```

Open daarna:

```text
http://localhost:8080
```
